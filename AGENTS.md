## Overview

Screen mirror & Android ADB Tool untuk macOS. Gaming Edition dengan:
- Screen mirroring, Game Mode, Performance Monitor
- **🆕 File Browser** — navigasi folder Android, upload/download/delete/rename
- **🆕 Tabbed UI** — STREAM + FILES tabs
- **🆕 argparse** — `--version` dan `--help` tanpa GUI

Merge progress dari joy-connect-for-mac (Swift → Python/PyQt5).
